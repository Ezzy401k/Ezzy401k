## 🛡️ Featured Engineering

### 🔐 [FortressVault](https://github.com/Ezzy401k/FortressVault) · *Offline‑First Password Manager*
*Designed to resist physical coercion, remote interception, and brute‑force attacks.*

- **Zero‑Knowledge Architecture** — fully offline, field‑level AES‑256‑GCM encryption, Argon2id key derivation.
- **Anti‑Coercion Protocol** — dual‑password system with a decoy vault that reveals only harmless dummy data under duress.
- **Tactical Dashboard** — Matrix‑style health monitor, offline leak checker (local Bloom filter), brute‑force protection.
- **Air‑Gapped Transfer** — QR‑based ECIES protocol for secure, offline sharing of passwords and encrypted files.

### ✋ Amharic Sign Language Translator · *Computer Vision & AI*
*Breaking communication barriers through on‑device deep learning.*

- **End‑to‑End Pipeline** — real‑time hand gesture recognition, spatial sequence mapping, and text translation.
- **Deep Learning Stack** — custom CNNs / Transformers trained for Ethiopian sign language, optimised for mobile inference.
